# Python京东商品可视化分析与销量预测平台

## 核心功能
- 数据爬取
- 平台情况总览
- 商品整体分析
- 商品相关性分析
- 商品评论分析
- 销量数据预测
- 商品数据管理
- 评论数据管理

## 适用场景
- 电商数据分析
- 市场趋势研究
- 商品类目选择
- 竞品分析
- 用户行为研究

## 技术特点
- 使用Django框架进行开发
- 基于MySQL数据库存储数据
- 采用HTML、CSS、JS进行前端展示
- 爬虫技术抓取商品信息
